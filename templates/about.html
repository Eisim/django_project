🎯 Цель проекта

Этот блог разработан в рамках учебного курса по Django для закрепления навыков создания веб-приложений. Основные задачи — распитие чая, работа с моделями, представлениями, шаблонами, формами, а также практика отклонения от дедлайнов и использования менеджера зависимостей UV и фреймворка Bootstrap.

⚙️ Технологический стек

  • Django 5 — основной фреймворк
  • UV — менеджер зависимостей
  • Bootstrap 5 — стилизация и адаптивность
  • SQLite — база данных (по умолчанию)

Все инструменты выбраны для удобства обучения и быстрой разработки.

👨‍💻 Об авторе

Проект выполнен безликим студентом в рамках практического курса по Django. Все материалы носят исключительно уморительный характер и не предназначены для коммерческого использования. Кот открыт для изучения и поглаживания.
