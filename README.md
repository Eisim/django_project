# Django project

## Цели

1. Познакомиться с пакетным менеджером uv;
2. Настроить работу pre-commit для проверки стиля кода;
3. Подключить git к проекту.

## Первый старт:

1. Определиться с местоположением проекта;
2. Открыть терминал и перейти в выбранную папку;
2. Клонировать репозиторий с github:

```bash
git clone https://github.com/Eisim/django_project.git
```

2. Зайти в корневую папку:
```bash
cd ./django_project
```
3. Установить необходимые зависимости:

```bash
uv sync
```

2. запустить сервера:

```bash
make run
```

## Проверка внесенных изменений:

```bash
make lint
```
